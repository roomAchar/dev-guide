## 快速开始
> 获取代码
```bash
git clone git@192.168.0.1:dev-guide-front-end.git
```

> 生成本地配置文件,需要自行修改 .env 文件
```
cp .env.bak .env
```
> 安装依赖
```
npm install
```
> 运行项目
```
npm run serve
```
> 打包
```
npm run build
```
> 运行测试
```
npm run test
```
